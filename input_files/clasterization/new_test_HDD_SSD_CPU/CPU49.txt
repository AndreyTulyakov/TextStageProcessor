Заключение

По диаграмме среднегеометрической производительности процессоров в двенадцати играх видно, что в номинальном режиме первую строчку заняла модель Core i7-9700F. После "разгона" оба участника тестирования показали равные результаты.

Процессоры Core i7-9700F и Core i5-9600KF обеспечили комфортную производительность в разрешениях 1920 х 1080 и 2560 х 1440 во всех играх без исключения. Это является несомненным плюсом для них.

По соотношению цена/производительность вне конкуренции был процессор Core i5-9600KF. В зависимости от режима работы и разрешения он был выгоднее для покупки, чем модель Core i7-9700F, на 27-35%.

Итак, в данном тестировании лидер по производительности так и не был выявлен. Оба процессора показали в целом равные результаты. Однако проблема восьмиядерной модели заключалась в ее высокой стоимости. В результате процессор Core i7-9700F уступил своему шестиядерному сородичу в привлекательности покупки.

В связи с вышесказанным можно смело утверждать, что на сегодняшний день процессор Core i5-9600KF является более выгодным приобретением для игрового ПК, чем его старший родственник Core i7-9700F.
