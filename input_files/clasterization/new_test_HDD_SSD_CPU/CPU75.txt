Тестирование процессора Intel Core i5-9400F в двадцати играх 
Вступление

В данном обзоре будет изучена производительность нового процессора Intel, представляющего семейство Coffee Lake Refresh – Core i5-9400F.

Его соперниками стали следующие модели::

    Core i5-9600K;
    Core i5-8600K;
    Core i5-8600;
    Core i5-8400;
    Core i3-8350K;
    Core i5-7600K;

    Ryzen 7 2700X;
    Ryzen 7 2700;
    Ryzen 5 2600X;
    Ryzen 7 1800X;
    Ryzen 7 1700X.

Напомним, что о работе тестовых стендов, методике и обработке результатов можно узнать из подробного рассказа о тестировании комплектующих в играх.

Тестовая конфигурация

Тесты проводились на следующем стенде:

    Материнская плата №1: ASRock Fatal1ty Z370 Gaming K6, LGA 1151v2;
    Материнская плата №2: ASUS ROG STRIX Z270H GAMING, LGA 1151;
    Материнская плата №3: Gigabyte X470 AORUS Gaming 5, АМ4;
    Система охлаждения CPU: Corsair Hydro Series H105 (~1300 об/мин);
    Видеокарта: GeForce RTX 2080 Ti 11264 Mбайт - 1545/14000 (ASUS);
    Оперативная память №1: 2 x 8 Гбайт DDR4 Kingston HyperX Savage (Spec: 3000 МГц / 15-17-17-32-1t / 1.35 В), X.M.P. - on;
    Оперативная память №2: 2 x 8 Гбайт DDR4 Corsair Vengeance LPX (Spec: 2933 МГц / 15-17-17-32-1t / 1.35 В) , X.M.P. - on;
    Дисковая подсистема №1: 64 Гбайт, SSD ADATA SX900;
    Дисковая подсистема №2: 1 Тбайт, HDD Western Digital Caviar Green (WD10EZRX);
    Блок питания: Corsair HX850 850 Ватт (штатный вентилятор: 140-мм на вдув);
    Корпус: открытый тестовый стенд;
    Монитор: 27" ASUS PB278Q BK (Wide LCD, 2560x1440 / 60 Гц).

Инструментарий и методика тестирования

Для более наглядного сравнения процессоров все игры, используемые в качестве тестовых приложений, запускались в разрешении 1920 х 1080.

В качестве средств измерения быстродействия применялись встроенные бенчмарки, утилиты FRAPS 3.5.9 Build 15586 и AutoHotkey v1.0.48.05. Список игровых приложений:

    Assassin's Creed Odyssey.

Во всех играх замерялись минимальные и средние значения FPS. В тестах, в которых отсутствовала возможность замера минимального FPS, это значение измерялось утилитой FRAPS. VSync при проведении тестов был отключен.

Заключение

По диаграмме производительности процессоров в двадцати играх видно, что Core i5-9400F занял третье место в обоих режимах работы, разделив его с Core i5-8600.

Прирост производительности Core i5-9400F по сравнению с предшественником Core i5-8400 составил гомеопатические 1-2%. В принципе, в этом нет ничего неожиданного, так как тактовые частоты новичка в обоих режимах работы увеличились на 100 МГц.

Несомненным плюсом для Core i5-9400F стало то, что он показал высокие результаты во всех протестированных играх.

В такой важной дисциплине, как соотношение цена/производительность, новичок компании Intel разделил вторую позицию со своим предшественником Core i5-8400.

Основываясь на итогах проведенного тестирования, можно утверждать, что компания Intel незначительно укрепила свои позиции на рынке процессоров, выпустив модель Core i5-9400F. Новый процессор ничем не выделяется на фоне своего предшественника Core i5-8400, однако, при этом был быстрее всех старших моделей компании AMD.

Также стоит отметить, что компания Intel наконец-то начала снижать стоимость своей продукции. Как следствие, новый процессор Core i5-9400F оказался привлекательным для покупки, заметно опередив по соотношению цена/производительность конкурирующую модель Ryzen 5 2600X.
