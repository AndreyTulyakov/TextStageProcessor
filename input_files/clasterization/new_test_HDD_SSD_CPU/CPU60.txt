Core i5-8400 VS Core i3-8350K – тесты в 1080p и 1440p 
Вступление

В данном материале будет рассмотрена производительность процессоров Core i5-8400 и Core i3-8350K в разрешениях 1920х1080 и 2560х1440.

Посмотрим, кто выйдет победителем из этого противостояния.



Тестовая конфигурация

Тесты проводились на следующем стенде:

    Материнская плата: ASRock Fatal1ty Z370 Gaming K6, LGA 1151v2;
    Видеокарта: GeForce RTX 2080 Ti 11264 Mбайт - 1545/14000 МГц (ASUS);
    Система охлаждения CPU: Corsair Hydro Series H105 (~1300 об/мин);
    Оперативная память: 2 x 8 Гбайт DDR4 Kingston HyperX Savage (Spec: 3333 МГц / CL15 / 1.35 В) , X.M.P. - on;
    Дисковая подсистема №1: 64 Гбайт, SSD ADATA SX900;
    Дисковая подсистема №2: 1 Тбайт, HDD Western Digital Caviar Green (WD10EZRX);
    Блок питания: Corsair HX850 850 Ватт (штатный вентилятор: 140-мм на вдув);
    Корпус: открытый тестовый стенд;
    Монитор: 27" ASUS PB278Q BK (Wide LCD, 2560x1440 / 60 Гц).

Процессоры:

    Core i5-8400 - 2800 @ 3800 МГц;
    Core i3-8350K - 4000 @ 4900 МГц.

Программное обеспечение:

    Операционная система: Windows 10 x64;
    Драйверы видеокарты: NVIDIA GeForce 430.64 WHQL.
    Утилиты: D3DGear 5.00 Build 2252, AutoHotkey v1.0.48.05, MSI Afterburner 4.6.0.

Заключение

По диаграмме среднегеометрической производительности процессоров в пятнадцати играх видно, что в штатном режиме процессор Core i5-8400 опередил Core i3-8350К на 5-6%. После разгона обе модели показали равные результаты. Модели Core i5-8400 и Core i3-8350К обеспечили комфортную производительность в разрешениях 1920х1080 и 2560х1440 во всех играх без исключения. Это является большим плюсом для них.

По соотношению цена/производительность безусловным лидером был процессор Core i3-8350К.

Основываясь на результатах данного тестирования можно утверждать: за счет обоих режимов работы победителем из схватки вышел процессор Core i5-8400. Правда, его превосходство над Core i3-8350К было минимальным.

Теперь давайте рассмотрим привлекательность покупки обоих моделей.

Если пользователь планирует сэкономить и купить процессор на ближайшие два-три года, то ему однозначно подойдет Core i3-8350К. Некоторое время спустя, его можно будет заменить, на старшую шести или восьми ядерную модель.

В случае покупки процессора на ближайшую пятилетку, лучше немного переплатить и приобрести процессор Core i5-8400. Уже в текущих играх он практически ни в чем не уступает разогнанному Core i3-8350К, работающему на тактовой частоте 4900 МГц. С учетом того, что современные игры начали массово оптимизироваться под многоядерные процессоры, нет никаких сомнений, что в обозримом будущем Core i5-8400 будет быстрее Core i3-8350К.
