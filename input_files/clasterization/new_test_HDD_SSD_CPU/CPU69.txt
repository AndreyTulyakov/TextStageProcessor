Тестирование процессоров Intel Core i7-9700F, Core i5-9600, Core i5-9500F и Core i3-9100F в 15 играх Вступление

В данном обзоре будет изучена производительность новых процессоров Intel Core i7-9700F, Core i5-9600, Core i5-9500F и Core i3-9100F. В обзоре присутствуют сводные диаграммы производительности ЦП и среднегеометрические результаты CPU в пятнадцати играх. Статья носит справочный характер, комментарии отсутствуют, поскольку каждый читатель сможет самостоятельно почерпнуть нужную ему информацию.

Напомним, что с работой тестовых стендов, методикой и обработкой результатов тестирования можно ознакомиться, перейдя по этой ссылке.
