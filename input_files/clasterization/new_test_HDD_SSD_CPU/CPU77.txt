Заключение

По диаграмме среднегеометрической производительности CPU в восьми играх видно, что в данном тесте модели Intel были заметно быстрее, чем их извечные конкуренты. Тем не менее, это не помешало процессорам AMD продемонстрировать высокие результаты.

В штатном режиме работы первую строчку заняли Core i7-8700K и Core i7-8700, вторую – Core i5-8600К, Core i7-7700K и Core i5-8600, третью – Core i7-7700 и Ryzen 7 2700X. После разгона расстановка сил претерпела незначительные изменения. Нешуточная борьба за лидерство развернулась между Core i7-8700K, Core i5-8600К, Core i7-8700 и Core i7-7700K, а вслед за ними расположились Core i5-8600, Ryzen 7 2700X и Ryzen 7 2700.

По соотношению цена/производительность в номинальном режиме работы лидирующую позицию занял процессор Core i5-8600. Вторая строчка осталась за моделью Core i5-8600К, а третье место «оккупировал» ЦП Ryzen 7 2700. После разгона расклад сил особо не изменился.

По итогам теста можно рекомендовать для покупки Core i5-8600 и Ryzen 7 2700. Пусть они и не стали самыми быстрыми CPU в этой игровой подборке, однако без затруднений обеспечили комфортную производительность, да и от Core i7-8700K отстали незначительно. Среди прочих плюсов отметим умеренную цену. В сочетании с высокой производительностью данный фактор сделал оба процессора наиболее привлекательными.
