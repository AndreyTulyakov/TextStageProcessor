     В основе формования химических волокон лежит растяжение струй расплавов и растворов полимеров. Известен подход к математическому описанию формования химических волокон, основанный на ряде допущений для представления  элонгационного течения тонкой струи путем усреднения по радиусу [1], где из уравнения Навье-Стокса и краевых условий получено одномерное стационарное уравнение движения. Эквивалентный результат получен с использованием баланса сил, действующих на струю при одномерном стационарном движении [2, 3] 
     
     Frh(x) = Frh(0) + Fsf(x) + Fin(x) + F?r(x) - Fgr(x)				(1)

     Эта связанная система обыкновенных дифференциальных уравнений (9-10) является базовой редуцированной системой уравнений стационарного элонгационного течения неизотермической струи расплава полимера. Распределение температуры по длине нити влияет на ее продольную вязкость и, соответственно, на ее скорость. Вместе с тем скорость нити связана с распределением температуры через коэффициент теплоотдачи. Следует отметить, что возможно только численное решение этой системы уравнений из-за ее существенной нелинейности.
     Для оценки работоспособности предлагаемой редуцированной системы уравнений (9-10), описывающей элонгационное течение струй полимеров, были проведены расчеты для случая, когда известны экспериментальные и расчетные данные, полученные в работе [4]. Рассматривалось течение расплава полипропилена при финальной скорости струи vf = 500 м/мин. Параметры и условия течения, свойства полимера были взяты из работы [4]. Начальные условия для решения уравнения движения (9) определялись по итерационной схеме, когда по известной финальной скорости находилось значение продольного градиента скорости dv/dx при x = 0. Были рассчитаны площади поперечного сечения струи и ее температуры в зависимости от расстояния от фильеры x.
